## Títulos

Se usan mejor para encabezados de sección.

```md
## Titles
```

### Subtítulos

Se usan mejor para encabezados de subsección.

```md
### Subtitles
```

<Tip>

Cada **título** y **subtítulo** crea un ancla y también aparece en la tabla de contenidos a la derecha.

</Tip>

## Formato de Texto

Soportamos la mayoría del formato markdown. Simplemente agrega `**`, `_`, o `~` alrededor del texto para formatearlo.

| Estilo        | Cómo escribirlo   | Resultado       |
| ------------- | ----------------- | --------------- |
| Negrita       | `**bold**`        | **bold**        |
| Cursiva       | `_italic_`        | _italic_        |
| Tachado       | `~strikethrough~` | ~strikethrough~ |

Puedes combinar estos. Por ejemplo, escribe `**_bold and italic_**` para obtener texto **_bold and italic_**.

Necesitas usar HTML para escribir texto en superíndice y subíndice. Es decir, agrega `<sup>` o `<sub>` alrededor de tu texto.

| Tamaño de Texto | Cómo escribirlo          | Resultado              |
| --------------- | ------------------------ | ---------------------- |
| Superíndice     | `<sup>superscript</sup>` | <sup>superscript</sup> |
| Subíndice       | `<sub>subscript</sub>`   | <sub>subscript</sub>   |

## Enlazar a Páginas

Puedes agregar un enlace envolviendo el texto en `[]()`. Escribirías `[enlace a google](https://google.com)` para [enlazar a google](https://google.com).

Los enlaces a páginas en tu documentación deben ser relativos a la raíz. Básicamente, debes incluir toda la ruta de carpetas. Por ejemplo, `[enlace a texto](/en/writing-content/text)` enlaza a la página "Text" en nuestra sección de componentes.

Los enlaces relativos como `[enlace a texto](../text)` se abrirán más lentamente porque no podemos optimizarlos tan fácilmente.

## Citas en bloque

### Una sola línea

Para crear una cita en bloque, agrega un `>` delante de un párrafo.

> Dorothy la siguió a través de muchas de las hermosas habitaciones de su castillo.

```md
> Dorothy la siguió a través de muchas de las hermosas habitaciones de su castillo.
```

### Múltiples líneas

> Dorothy la siguió a través de muchas de las hermosas habitaciones de su castillo.
>
> La Bruja le ordenó limpiar las ollas y teteras, barrer el suelo y mantener el fuego alimentado con leña.

```md
> Dorothy la siguió a través de muchas de las hermosas habitaciones de su castillo.
>
> La Bruja le ordenó limpiar las ollas y teteras, barrer el suelo y mantener el fuego alimentado con leña.
```

### LaTeX

Mintlify soporta [LaTeX](https://www.latex-project.org) a través del componente Latex.

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
